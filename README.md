# Neovim Setup

Neovim using lua configuration has the following directory structure:

```
~/.config/nvim
|-- lua/
|   |-- config/
|       |-- plugins/
|       |-- init.lua
|       |-- lazy_init.lua
|       |-- remap.lua
|       |-- set.lua
|-- README.md
|-- init.lua
|-- lazy-lock.json
```

My personal Neovim setup with lazy and lua.
